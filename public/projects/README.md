# Project screenshots

Add screenshot images here (e.g. `sibme.png`, `fitflow.png`), then in `src/components/Projects.tsx` set the `image` field for that project:

```ts
{
  title: "Sibme",
  image: "projects/sibme.png",
  imageAlt: "Sibme dashboard",
  // ...
}
```

Use landscape images (e.g. 1200Ã—630) for best results. Next.js will optimize them automatically.
